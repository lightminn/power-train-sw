## 전류 관련 두 가지 구분

이 둘은 헷갈리기 쉬운데 다른 거다.

- **Current Limit** (control table item 38, EEPROM) — 모든 모드에서 적용되는 전류 상한선. 안전장치 역할.
- **Goal Current** (item 102, RAM) — Current Control 모드 또는 Current-based Position 모드에서 실제 목표 전류값.

읽기는 **Present Current** (item 126)로 한다. 단위는 모델별 current scaling factor가 다르므로 (XM430-W350은 약 2.69mA/unit) datasheet 확인 필요.

I need to clarify what stall current actually means for these motors—it's the current drawn when the motor is locked at maximum torque, not a hard limit on current draw. The XM430-W350 specs show 1.4A at stall, but there's a key misconception to address here about whether that's truly the maximum the motor will ever draw.

The Current Limit register can actually be set higher than the stall current value—it goes up to about 3.21A on this model, well above the 1.4A stall rating. So if I set the Current Limit to 2A, the motor controller will try to maintain that even though it exceeds the stall current specification. The motor can briefly exceed stall current, and sustained high current causes heating regardless of whether it's technically at or below the stall rating. There's also the Shutdown register that provides additional hardware protection through temperature monitoring and error detection, so the motor won't just keep drawing unlimited current indefinitely.엄밀히 말하면 *Current Limit을 어떻게 설정하느냐*에 달렸다. 자동으로 stall 전류 막아주는 게 아니다.

Current Limit (item 38)은 사용자가 설정하는 값이고, 모델별 최댓값까지 올릴 수 있다. 예를 들어 XM430-W350은:
- Stall current (정격): 약 1.4A @ 12V
- Current Limit 설정 가능 범위: 0 ~ 1193 units (약 3.21A)

즉 Current Limit을 풀로 열어두면 stall 전류의 2배 넘게도 흐를 수 있다. 짧은 시간 동안은. 그래서 보수적으로 Current Limit을 stall current 근처 또는 그 이하로 잡아두는 게 일반적이다.

그리고 stall current 이하라도 안심하면 안 되는 게 — stall은 *정격* 전류지 *안전* 전류가 아니다. 1.4A를 계속 흘리면 코일 발열로 모터 타들어간다. 데이터시트의 stall torque/current는 "이 이상 토크는 못 낸다"는 한계지, "여기까지는 막 써도 된다"는 게 아님.

다이나믹셀 자체적인 추가 보호 장치도 있다:

- **Shutdown** (item 63): Overload, Overheating, Input Voltage, Electrical Shock 같은 에러 발생 시 자동으로 토크 끄는 비트마스크. 디폴트로 거의 다 켜져 있다.
- **Temperature Limit** (item 31): 보통 80°C 디폴트. 넘으면 Shutdown에 따라 토크 차단.
- **Overload error**: 정격 토크 초과한 상태가 일정 시간 지속되면 트리거.

그래서 실전에선:

1. Current Limit을 stall current의 70~80% 정도로 잡는다 (예: XM430이면 1.0A 전후).
2. Shutdown 비트는 디폴트 그대로 두거나 모두 활성화.
3. Hardware Error Status (item 70) 주기적으로 polling해서 에러 잡아냄.
4. 그래도 발열 모니터링은 별도로 — Present Temperature (item 146) 읽어서 70°C 넘으면 동작 줄이는 로직.
