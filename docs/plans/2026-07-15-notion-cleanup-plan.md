## P4 — 템플릿 정합 (섹션 수술)

표준: 개요 콜아웃→목차→①환경→②핵심개념→③배선(HW시)→④설치→⑤실행→⑥트러블슈팅→⑦검증→⑧코드·참고
(미사용 절 삭제+순차 재번호 허용).

**전 페이지 공통 수락 기준 — 복붙 완주 (2026-07-15 사용자 재강조):**
모든 절차 페이지는 **처음 읽는 사람이 스크립트 복붙만으로 그 페이지가 설명하는 것을 실행
완료**할 수 있어야 한다. 판정 기준: ① SSH 접속 직후 홈(`~`) 기준 시작 — 레포 이동→호스트
준비(can_setup 등)→컨테이너 진입→실행까지 빠짐없이, 중간 상태(컨테이너 떠 있음·CAN 올라옴)
가정 금지 ② 명령 블록마다 **어디서**(노트북/호스트/컨테이너/odrivetool — odrivetool이면 켜는
법부터) + **✅ 기대 출력** 명시 ③ 참조 스크립트는 전부 레포에 존재(레포 외부 파일 인용 금지).
P1~P4 각 단계에서 손대는 페이지마다 이 기준으로 재검사한다. 감사 기준 현재 미달 확실:
「L515 1차 검증」(레포 외부 `l515_zone_logger.py`, Windows venv 환경), 「통신 스트리밍 GUI
분석」(코드 절 미연결) — 둘 다 📜 격하 대상이라 격하 콜아웃에 미달 사실 명시로 갈음.
